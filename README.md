# upskill-Campus: Python URL Shortener

This repository contains the final project for my **6-week Industrial Internship** at **UniConverge Technologies Pvt. Ltd. (UCT)**, facilitated by **Upskill Campus (USC)**.

## Project Overview
The **Python URL Shortener** is a Flask-based web application designed to convert long, cumbersome URLs into compact, shareable identifiers.

## Tech Stack
* **Language:** Python 3.11
* **Web Framework:** Flask
* **Database:** SQLite3
* **Frontend:** HTML5 (Jinja2 Templates)

## Repository Structure
* `app.py`: The main application logic and Flask routes.
* `urls.db`: The SQLite database for storing URL mappings.
* `Amit_Sharma_Internship_Report.pdf`: The detailed technical internship report.

## How to Run
1. Install Flask: `pip install flask`
2. Run the application: `python app.py`
3. Access locally at: `http://127.0.0.1:5000`
